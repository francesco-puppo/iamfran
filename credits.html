Originally, I built this site in Webflow but since I stripped it down to only one page, I preferred to code it from scratch using VS Code and host it on GitHub.

For the text and the social icons entry animations I used Animista.

The social icons come from FontAwesome: their hover effect is a modified version of this Codepen here, while the tooltip comes from here.

Figma was used to design the mock-up and to test accessibility with Stark.
